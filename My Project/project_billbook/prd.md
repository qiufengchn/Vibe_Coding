# 项目账本应用 PRD

## 1. 产品概述
一个专注于项目级别收支管理的账本应用，适用于装修、项目投资等大额特定场景的财务记录。采用现代化的 UI 设计，提供流畅的用户体验。

## 2. 核心功能

### 2.1 账本管理
- 创建新账本
  - 账本名称
  - 创建日期
  - 账本描述（可选）
  - 预算金额（可选）
- 账本列表展示
  - 卡片式布局
  - 显示账本名称、总收支、创建日期
  - 支持删除账本
  - 支持编辑账本信息

### 2.2 收支记录
- 新增记录
  - 类型：收入/支出
  - 金额
  - 日期
  - 分类（可自定义）
  - 备注
  - 支付方式
- 记录列表
  - 按日期倒序排列
  - 支持筛选（按类型、日期范围、分类）
  - 支持搜索
  - 支持编辑和删除

### 2.3 数据统计
- 收支总览
  - 总收入
  - 总支出
  - 结余
- 分类统计
  - 饼图展示各类别占比
  - 柱状图展示收支趋势

## 3. 技术方案

### 3.1 前端技术栈
- React + TypeScript
- TailwindCSS 用于样式
- IndexedDB 用于本地数据存储
- Chart.js 用于数据可视化
- Framer Motion 用于动画效果

### 3.2 数据存储结构
```typescript
interface Book {
  id: string;
  name: string;
  description?: string;
  budget?: number;
  createdAt: Date;
  updatedAt: Date;

interface Transaction {

  bookId: string;
  type: 'income' | 'expense';
  amount: number;

  category: string;
```

## 4. UI/UX 设计

### 4.1 配色方案
- 主色调：优雅的深蓝色 (#1E3A8A)
- 强调色：清新的绿色 (#059669)
- 背景色：柔和的灰白色 (#F9FAFB)
- 渐变效果：
  - 主按钮：深蓝到浅蓝渐变
  - 收入按钮：绿色渐变
  - 支出按钮：红色渐变

### 4.2 布局设计
- 响应式设计，支持移动端和桌面端
- 左侧固定导航栏
- 主内容区采用卡片式布局
- 重要操作按钮使用渐变背景
- 数据可视化区域采用卡片式布局

### 4.3 交互设计
- 所有操作添加平滑过渡动画
- 表单提交添加加载动画
- 数据更新时添加优雅的过渡效果
- 支持拖拽排序
- 支持手势操作（移动端）

## 5. 性能优化
- 使用 IndexedDB 的索引优化查询性能
- 实现数据分页加载
- 使用 React.memo 优化组件重渲染
- 实现数据缓存机制
